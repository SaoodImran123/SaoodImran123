# Resume Website

Personal portfolio website for Saood Imran.

## Development

1. Clone the repository:
   ```bash
   git clone https://github.com/SaoodImran123/ResumeWebsite.git
   cd ResumeWebsite
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
